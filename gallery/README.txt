Club photos for the website
---------------------------
1. Copy your images (JPG, PNG, WebP, or GIF) into this folder on your PC.
2. In PowerShell, from the project folder, run:
     .\update-gallery-manifest.ps1
   This updates manifest.json so the site knows which files to show.
3. Commit and push to GitHub so Vercel deploys the new photos.

Tip: Name files in the order you want (e.g. 01-match.jpg, 02-team.jpg).
